# 医院管理信息系统 java441

## 项目概述

医院管理信息系统是基于 springboot 和 vue 框架开发的门诊前后台管理系统，支持挂号预约等管理功能。

## 技术栈

- **前端技术**：Vue2 + ElementUl
- **后端技术**：SpringBoot2 + MyBatisPlus + Redis
- **数据库**：MySQL
- **编程语言**：Java

## 功能模块

### 系统角色

- 管理员
- 医生
- 患者

### 管理员功能

- **登录**
- **医生信息管理**
- **患者信息管理**
- **挂号信息管理**
- **药物信息管理**
- **检查项目管理**
- **病床信息管理**
- **排班信息管理**
- **数据统计分析**

### 医生功能

- **登录**
- **首页**
- **今日挂号列表**
- **历史挂号列表**
- **住院申请管理**
- **个人信息查询**

### 患者功能

- **注册登录**
- **首页**
- **预约挂号**
- **我的挂号**
- **住院信息**
- **个人信息**

## 运行环境

- 前端：支持现代浏览器，推荐使用 Chrome、Firefox 或 Safari
- 后端：Java 1.8 或更高版本，MySQL 5.7 或更高版本，Redis 3.2 或更高版本

## 部署步骤

1. 准备所需软件环境：Java, MySQL, Redis
2. 导入数据库脚本
3. 部署后端服务
4. 部署前端服务
5. 启动系统
